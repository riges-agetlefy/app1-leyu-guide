# app1-leyu-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或具体网站，仅作为通用前端页面的存储与展示空间。

仓库名称中的“leyu-guide”为归档标识，不代表任何特定含义或指向。

## 目录结构

```
app1-leyu-guide/
├── index.html          # 仓库首页（可选）
├── pages/              # 存放归档的 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：归档的独立 HTML 页面，每个页面可独立访问。
- `assets/`：页面所需的外部资源文件，按类型分目录存放。

## 页面归档说明

归档的 HTML 页面均为静态文件，不依赖后端服务。每个页面均可在浏览器中直接打开运行，或通过 GitHub Pages 等服务进行托管访问。

页面内容可能包含演示、工具、信息展示等，不涉及具体商业或营销用途。归档仅用于技术展示与资料保存。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件。
3. 或通过 GitHub Pages 功能启用访问（需自行配置）。
